
# <こみこみゅ>

## サイト概要
### サイトテーマ
好きな漫画の紹介をできるコミュニティサイト

### テーマを選んだ理由
自分自身が漫画が好きでよく読むのですが、色々な人のおすすめの漫画をしれたらいいなと思ったから。
また、漫画が好きな人と交流できるサイトがあったらいいなと思ったからこのテーマにしました。

### ターゲットユーザ
漫画が好きな人、興味がある人

### 主な利用シーン
自分が読んだ漫画を紹介したいとき
